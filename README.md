# Vector
> Vector utilities for NLP - NodeJS and Browser

## Usage

```bash
npm i gafo-vector
```

```html
<script src="https://s3.amazonaws.com/gafo-vector/vector.js"></script>
<script>
  var v = new Vector();
</script>
```

```javascript
var Vector = require('gafo-vector');

var v = new Vector();
```
